# Assets Directory

This directory contains static assets used in media production.

## Structure

### backgrounds/
Background templates for thumbnails. Place your custom background images here.
- Recommended size: 1280×720 pixels
- Format: JPEG or PNG

### logos/
Church logo and branding assets.
- Format: PNG with transparency preferred
- Keep multiple sizes for different use cases

### pastor/
Pre-made pastor portrait assets for thumbnails.
- Format: PNG with transparency preferred
- Recommended size: 200×200 to 400×400 pixels

### fonts/
Custom fonts for consistent typography in thumbnails.
- Include both regular and bold weights
- TTF or OTF format

## Usage

These assets are referenced by the thumbnail composer module to create consistent, branded thumbnails for each sermon.

## Adding Assets

1. Place new files in the appropriate subdirectory
2. Update your event configuration or global config to reference them
3. Ensure proper permissions for the workflow to read these files
